# Ausgabe-Bildformate

### TIFF (16-Bit)

16-Bit (Pixel-Digitalzahl 0–65535) Ausgabe im Format TIFF (.tif).

### TIFF (32 Bit, Prozent %)

32 Bit (Pixel Float 0,0–1,0) Ausgabe im Format TIFF (.tif). Die Pixelwerte stehen für Reflexionsprozentsätze, wobei 0,0 = 0 % und 1,0 = 100 % entspricht.

### **JPG (8 Bit)**

8 Bit (Pixel-Digitalzahl 0–255) Ausgabe im JPG-Format (.jpg).

### **PNG (8 Bit)**

8-Bit (Pixel-Digitalzahl 0–255) PNG (.png)-Formatausgabe. Verlustfreie Komprimierung, geeignet für die Anzeige und Visualisierung im Internet.
