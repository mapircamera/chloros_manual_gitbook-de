# Erste Schritte

<div data-full-width="false"><figure><img src=".gitbook/assets/chloros_logo_transparent.png" alt=""><figcaption></figcaption></figure></div>Chloros ist eine Softwareanwendung von [MAPIR](https://www.mapir.camera) zur Verarbeitung von Bildern und anderen Sensordaten.

***

Chloros ist in 4 Anwendungsmodi verfügbar:

## Chloros: Desktop-GUI-Anwendung

\[Empfohlen] Eigenständiges separates Fenster mit allen Funktionen.

## Chloros (Browser): Webbasierte GUI-Schnittstelle

Läuft in Ihrem Webbrowser (Chrome/Edge). Am besten geeignet für langsamere Computer mit &lt;4 GB VRAM.

## [Chloros CLI: Befehlszeilenschnittstelle](CLI.md)

Befehlszeilen-Stapelverarbeitung. Perfekt für Automatisierung, Skripterstellung und erweiterte Workflows. _Für den Zugriff auf die CLI ist eine Chloros+-Lizenz erforderlich._

## [Chloros API: Python SDK](api-python-sdk.md)

Programmatische Python-Schnittstelle für Automatisierung und benutzerdefinierte Workflows. Perfekt für Forschungs-Pipelines, die Integration in bestehende Python-Anwendungen und die Erstellung benutzerdefinierter Tools. _Für den Zugriff auf die API ist eine Chloros+-Lizenz erforderlich._

***

## Chloros+

Chloros kann für die meisten Aufgaben kostenlos genutzt werden, aber vielleicht möchten Sie mehr. In diesem Fall kann Ihnen eine kostenpflichtige Lizenz für Chloros+ Vorteile bieten. Mit einer Chloros+-Lizenz können Sie neue Funktionen freischalten, darunter:

* **Multithread-Verarbeitung**: Beschleunigen Sie die Bildverarbeitung für größere Projekte erheblich, indem Sie Bilder gleichzeitig über die Pipeline verarbeiten.
* **GPU (CUDA)-Beschleunigung**: Nutzen Sie die Vorteile der heutigen höheren GPU-Speicheroptionen, um die Bildverarbeitungs-Pipeline weiter zu beschleunigen. Für optimale Ergebnisse empfehlen wir 4 GB oder mehr VRAM.
* **Chloros+** [**CLI**](CLI.md) **Zugriff**: Führen Sie Chloros+ über die Befehlszeile aus, um es zu automatisieren und in Ihre eigene Software zu integrieren.
* **Chloros+** [**API**](api-python-sdk.md) **Zugriff:** Führen Sie Chloros+ über Python aus, um eine programmatische Steuerung zu ermöglichen und eine nahtlose Integration in Ihre Forschungs-Pipelines, Datenanalyse-Workflows und benutzerdefinierten Anwendungen zu erreichen.
* **Nutzung auf mehreren Geräten**: Mit jeder Chloros+-Lizenz können zwei oder mehr Geräte registriert werden. Verwenden Sie Ihr MAPIR-Cloud-Konto, um registrierte Geräte zu verwalten. Fügen Sie Unterstützung für weitere Geräte hinzu, indem Sie Ihre Chloros+-Lizenz upgraden.
* **Benutzerdefinierte Multispektralindex-Formeln:** Geben Sie benutzerdefinierte Multispektralindizes in die Chloros-Rasterrechner ein, sowohl für die Verarbeitung als auch für die Bildbetrachtungssandbox.
