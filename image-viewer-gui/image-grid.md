# Bildraster

Nachdem Sie Bilder in ein Projekt importiert haben, werden diese im Hauptbereich in einem Raster angeordnet angezeigt.

## Größe der Miniaturansichten

Mit dem Zoom-Schieberegler oben rechts können Sie die Größe der Bildminiaturen anpassen. Sie können auch die STRG-Taste gedrückt halten und das Mausrad drehen, um sie zu skalieren.

## Schaltflächen zum Exportieren der Ansicht

Während die Bilder verarbeitet werden, werden oben links im Rasterbereich Schaltflächen angezeigt. Mit diesen Schaltflächen können Sie zwischen verschiedenen Bildtypen in der Rasteransicht wechseln:

* **JPG** – Original-JPG-Miniaturansichten der Kamera
* **Ziele** – Bilder, die erkannte Kalibrierungsziele enthalten
* **Reflexionsgrad** – Kalibrierte Reflexionsgradbilder
* **Index/LUT** – Berechnete multispektrale Indexbilder mit einem Farb-LUT-Farbverlauf

Durch Klicken auf die Miniaturansichten im Raster wird der Vollbildmodus [Image Viewer](opening-an-image-full-screen.md) geöffnet. Wenn der Anzeigemodus im Raster auf einen bestimmten Export wie „Targets” eingestellt ist, wird durch Klicken auf die Miniaturansicht im Raster das exportierte Zielbild im Vollbildmodus geöffnet.
