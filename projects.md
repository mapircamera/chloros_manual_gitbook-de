# GUI: Projekte

Mit Chloros können Sie Projekte erstellen, die später wieder geöffnet werden können.

## Neues Projekt

<figure><img src=".gitbook/assets/project_new.JPG" alt=""><figcaption></figcaption></figure>

Wählen Sie „Neues Projekt” aus dem Hauptmenü und geben Sie einen eindeutigen Namen für Ihr Projekt ein.

## Projekt öffnen

<figure><img src=".gitbook/assets/project_open.JPG" alt=""><figcaption></figcaption></figure>

Wählen Sie „Projekt öffnen”, um eine Liste der vorhandenen Projekte im Projektordner anzuzeigen. Wenn keine Projekte vorhanden sind, wird das sekundäre Seitenmenü nicht geöffnet. Auf dem obigen Foto sehen Sie einige mit der GUI erstellte Projekte (t1, t2, t3). Die Projekte DATE\_TIME wurden von CLI unter Verwendung des Standard-Projektnamensschemas erstellt. Durch Klicken auf einen Projektnamen wird dieses geöffnet.

Durch Klicken auf die Schaltfläche „Projektordner öffnen“ wird der Datei-Explorer Ihres Computers mit dem Projektpfad geöffnet. Sie können den Projektpfad in den [Projekteinstellungen](project-settings/project-settings.md) anpassen.
